# bookishelf
An app that uses open library api, where you can search and get information about books.

## bookishelf backend
1. Open a terminal, go to bookishelf-backend
2. `npm install`
3. `npm run build`
4. `npm run start`



## bookishelf frontend
1. Open a new terminal, go to bookishelf-frontend 

2. `npm install`
3. `npm run start`
